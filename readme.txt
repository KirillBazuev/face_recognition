Требуются: numpy, open-cv

В главном меню имеются три опции:

- Make new profile
	Собирает данные для обучения.
	При нажатии открывается окно с инструкцией, полем для ввода и кнопкой.
	В поле для ввода нужно ввести свое имя, и нажать кнопку "Continue"
	После этого откроется новое окно, которое берет изображение с веб-камеры, и отслеживает на ней лицо человека, периодически сохраняя кадры в папку myfaces
	Запись лица ведется на протяжении 20 секунд
- Recognition in real time
	Производит обучение на имеющихся кадрах из myfaces
	Включает веб-камеру, соотносит лицо человека с уже существующими профилями, при удачном распозновании выделяет и подписывает лицо
	Не работает при отсутствии профилей
- Delete all profiles
	Удаляет все профили


Распознавание происходит наилучшим образом, если при создании профилей и при самом распозновании у лиц был один и тот же фон
